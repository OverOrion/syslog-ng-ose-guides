Message format in for

The for is supported for Windows 7 or newer platforms. Starting with Windows 7, event logging is XML-based, meaning that event log messages reach in XML format. When these are forwarded to the server, parses them into key-value pairs using the XML parser.

Once event log data is available in , you have the flexibility to modify and format data any way you want, using macros and rewrite rules.

Note that while event log data as processed by the tool may differ from the data collected and made available by the , the tool provides you with greater freedom and flexibility when it comes to manipulating your raw data.
