Sudo parser

The sudo parser can parse the log messages of the sudo command. Available in version 7.0.93.16 and later.

Declaration:
@version: 
@include "scl.conf"
log {
    source { system(); };
    parser { sudo-parser(); };
    destination { ... };
};

The sudo-parser() is actually a reusable configuration snippet configured to parse sudo messages. For details on using or writing such configuration snippets, see more. You can find the source of this configuration snippet on GitHub.

By default, sudo-parser() uses the .sudo. prefix. To modify it, use the following format:

parser { 
    sudo-parser(prefix("myprefix.")); 
};
