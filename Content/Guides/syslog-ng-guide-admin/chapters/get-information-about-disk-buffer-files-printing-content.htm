Printing the content of disk-buffer files

Purpose

This section describes printing the content of the disk-buffer files used in ().

Command syntax

The command syntax for printing the content of the disk-buffer files used in looks like the following:

/opt/syslog-ng/bin/dqtooldqtool cat DISK-BUFFER_FILE
