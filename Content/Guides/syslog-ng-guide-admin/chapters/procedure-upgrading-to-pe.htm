Upgrading to

Purpose:

To upgrade to , complete the following steps:

Steps:

  1. Download the new installer package from &mybalabit;. Use the same package type as you used for the installation (for example, use the .run package for the upgrade if you have originally installed using a .run installer).

  2. Download the new license file from &mybalabit;.

  3. Install and check the warnings. Upgrade the respective parts of your configuration if needed.

  4. On the host where you are running in server mode, replace the old license file with the new one.

    		 Caution:

    Hazard of data loss! Without the new license file, will run in relay mode, and will not store the incoming messages locally.
