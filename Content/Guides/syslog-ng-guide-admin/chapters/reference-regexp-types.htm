Options of regular expressions

This chapter lists regular expressions supported by () and their available supported type() and flags() options.

By default, uses PCRE-style regular expressions. To use other expression types, add the type() option after the regular expression.

The application supports the following regular expression type() options:

  • Perl Compatible Regular Expressions (pcre)

  • Literal string searches (string)

  • Glob patterns without regular expression support (glob)
