The &abbrev; binaries include all required libraries and dependencies of &abbrev;, only the ncurses library is required as an external dependency (&abbrev; itself does not use the ncurses library, it is required only during the installation). The components are installed into the /opt/syslog-ng directory. It can automatically re-use existing configuration and license files, and also generate a simple configuration automatically into the /opt/syslog-ng/etc/syslog-ng.conf file.
