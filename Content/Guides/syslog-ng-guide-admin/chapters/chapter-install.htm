This chapter explains how to install on the supported platforms using the precompiled binary files.

The binaries include all required libraries and dependencies of , only the ncurses library is required as an external dependency ( itself does not use the ncurses library, it is required only during the installation). The components are installed into the /opt/syslog-ng directory. It can automatically re-use existing configuration and license files, and also generate a simple configuration automatically into the /opt/syslog-ng/etc/syslog-ng.conf file.
