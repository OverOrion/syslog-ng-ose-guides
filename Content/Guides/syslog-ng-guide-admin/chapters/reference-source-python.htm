python() and python-fetcher() source options

The python() and python-fetcher() drivers have the following options.

  • threaded: The threaded flag enables multithreading for the source. For details on multithreading, see Multithreading and scaling in [%=General.abbrev%].

    		 NOTE:

    The syslog source uses multiple threads only if the source uses the tls or tcp transport protocols.
