Installing in server mode

Purpose:

The following describes how to install on log servers. For details on the different operation modes of , see Modes of operation.

Steps:

To install on log servers

  1. Login to the Support Portal and download the installer package and your license file (license.txt). The license will be required to run in server mode (see Server mode) and is needed when you are installing on your central log server.

  2. Locating the license: Enter the path to your license file (license.txt) and select OK. Typically this is required only for your central log server.

    If you are upgrading an existing configuration that already has a license file, the installer automatically detects it.
