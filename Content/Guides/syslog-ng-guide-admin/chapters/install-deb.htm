Installing syslog-ng on Debian-based platforms

Purpose:

The following describes how to install syslog-ng on operating systems that use the Debian Software Package (deb) format. The following supported operating systems use this format:

  • Debian Jessie

  • Ubuntu 14.04 LTS (Trusty Tahr)

  • Ubuntu 16.04 LTS (Xenial Xerus)

  • Ubuntu 18.04 LTS (Bionic Beaver)

Steps:

To install syslog-ng on operating systems that use the Debian Software Package (deb) format

  1. Login to your account and download the DEB package for your system.

  2. Issue the following command as root:

    dpkg -i syslog-ng-premium-edition-<version>-<OS>-<arch>.deb

  3. Answer the configuration questions of . These are described in detail in Installing syslog-ng using the .run installer.
