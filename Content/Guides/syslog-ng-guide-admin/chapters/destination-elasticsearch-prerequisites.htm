Prerequisites

The following describes how to send messages from to Elasticsearch.

Steps:

To send messages from to Elasticsearch

  2. Download the Elasticsearch libraries version 1.5 or newer from the 1.x line from https://www.elastic.co/downloads/elasticsearch. tests the destination using Elasticsearch version 1.5. To use Elasticsearch 2.x or newer, use the elasticsearch2() destination (see elasticsearch2: Sending messages directly to Elasticsearch version 2.0 or higher).

  3. Extract the Elasticsearch libraries into a temporary directory, then collect the various .jar files into a single directory (for example, /opt/elasticsearch/lib/) where can access them. You must specify this directory in the configuration file. The files are located in the lib directory and its subdirectories of the Elasticsearch release package.
