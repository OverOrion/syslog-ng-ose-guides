Installing on Windows

Purpose:

The following instructions describe the standalone installation, which is configured locally. For more advanced installation options (using domain group policies, installing by group policy), refer to The Administrator Guide.

Steps:

  1. Execute the downloaded binary.

  2. Accept the EULA.

  3. Select the destination folder for .

  4. Choose Stand alone mode.

  5. The installer generates a simple configuration. Enter the destination IP of the server:

    1. Select Destinations

    2. Double-click Add new server

    3. Enter the server's IP address

    4. Change the port number to 601

    5. Click OK

  6. Close the configuration window to finish installation.

  7. Validating the installation

    Test remote logging:

    1. Log out and back in on the Windows client

    2. Verify the server log. On the server, enter the following command:

      tail /var/log/messages

    Expected outcome

    The logout and login events are displayed in the log.
