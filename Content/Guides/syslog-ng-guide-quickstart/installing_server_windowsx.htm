Downloading the server installer

Purpose:

To obtain the installer from &mybalabit;, complete the following steps:

Prerequisites:

The installers are available via &mybalabit;. In addition to the installers, a valid license is required to install the server. Contact your sales representative for access and license files.

Steps:

  1. Navigate to the and select .

  2. Choose the latest available version (5.0.2 is used as an example)

  3. Download the 32-bit or 64-bit installer, depending on your server's architecture:

    • For the 32-bit installer, download syslog-ng-premium-edition-5.0.2-win32.exe

    • For the 64-bit installer, download syslog-ng-premium-edition-5.0.2-win64.exe

    The binaries include all required libraries and dependencies of syslog-ng. These components are installed in the C:\Program Files\syslog-ng directory by default.

    The installer can reuse existing configuration and license files. Following installation, sample configuration files are also available in the etc subfolder.
