Configuring syslog-ng Linux clients for

Purpose:

The following describes how to configure the syslog-ng Linux clients for .

Steps:

To configure the syslog-ng Linux clients for

  1. Open the /opt/syslog-ng/etc/syslog-ng.conf configuration file in a text editor.

  2. Replace the line starting with destination d_logserver with the following:

    destination d_logserver {
    syslog("<PEServerIP>" transport(rltp())
        port("601")
        disk-buffer(disk-buf-size(2000000)));
};

    Replace <PEServerIP> with the hostname or IP address of the server.

  3. Save the file and restart syslog-ng.

    Remote logging can be tested the same way as described in Installing the [%=General.abbrev%] client on Linux.
