Downloading the Windows installer (client only)

Purpose:

The following describes how to obtain the installer from the product page on the Support Portal.

Prerequisites:

The installers are available through the product page on the Support Portal. In addition to the installers, a valid license is required to install the server. Contact your sales representative for access and license files.

Steps:

To obtain the installer from the product page on the Support Portal

  1. Navigate to the page.

  2. Choose the latest available version (6.0.3 is used as an example):

    • Release: 6 LTS

    • Component: syslog-ng Agent for Windows

    • Revision: 6.0.3

    • Platform: Windows 2012

  3. Select syslog-ng Agent with MMC snapin (setup) 64/32bit

  4. Download syslog-ng-agent-6.0.3-setup.exe

    Regardless of the path name, the installer contains both the 32-bit and the 64-bit binaries.
