Steps:

  • From the Start menu, launch the Configure syslog-ng Agent for Windows application

  • Select Destinations

  • Right-click the previously configured destination, and choose Properties

  • Enable RLTP

  • Choose OK to save your changes, and exit from the configuration interface

  • Restart syslog-ng Agent for the new configuration settings to take effect

    		NOTE:

    To restart services, you need Administrator privileges. If you use the Stop syslog-ng Agent and Start syslog-ng Agent options from the Start Menu, remember to right-click and choose the Run as Administrator option.
