syslog-ng Agent for Windows:

A lightweight client alternative to for Windows, the syslog-ng Agent for Windows application can collect and forward log messages to a remote server. It comes with a graphical user interface, and it's easier to deploy to a large number of machines.

Scope

This guide contains instructions for setting up (PE) as server and as client on Windows for evaluation.

In addition, basic configuration options are provided for reliable transfer protocol, macros in filenames, and storing messages in encrypted files.
