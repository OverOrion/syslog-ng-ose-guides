New features in 7.0.10

Send logs using the Advanced Log Transport Protocol ()

Using the new (), you can send (and receive) log messages in a reliable way over the TCP transport layer. is a proprietary transport protocol that prevents message loss during connection breaks. The transport is used between hosts (for example, a client and a server, or a client-relay-server), and interoperates with the flow-control and reliable disk-buffer mechanisms of , thus providing the best way to prevent message loss.

is the successor of the Reliable Log Transport Protocol (RLTP) introduced in version 6 LTS. Starting with version 7.0.9, the application can receive messages sent using RLTP from hosts that are running version 6 of or the application.
