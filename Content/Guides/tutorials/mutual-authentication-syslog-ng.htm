Configuring

Once you are ready with generating CA, server and client certificates, copy them to the respective machines and configure to use them. In theory, the CA and other certificates could be placed anywhere in the file system. In practice, server applications, such as are often protected by AppArmor, SELinux or other mechanisms, therefore it is recommended to create sub-directories where the syslog-ng.conf resides. This way can read them without modifying the related access rules.
