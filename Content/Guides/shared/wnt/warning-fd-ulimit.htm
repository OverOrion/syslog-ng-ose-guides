Caution:

When creating several thousands separate log files, () might not be able to open the required number of files. This might happen for example, when using the ${HOST} macro in the filename while receiving messages from a large number of hosts. To overcome this problem, adjust the --fd-limit command-line parameter of or the global ulimit parameter of your host. For setting the --fd-limit command-line parameter of see the The syslog-ng manual page. For setting the ulimit parameter of the host, see the documentation of your operating system.
