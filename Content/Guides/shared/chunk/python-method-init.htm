init(self, options) method (optional)

The application initializes Python objects only when it is started or reloaded. That means it keeps the state of internal variables while is running. The init method is executed as part of the initialization. You can perform any initialization steps that are necessary for your source to work.

When this method returns with False, does not start. It can be used to check options and return False when they prevent the successful start of the source.

options: This optional argument contains the contents of the options() parameter of the configuration object as a Python dict.
