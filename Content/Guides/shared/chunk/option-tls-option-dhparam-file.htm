dhparam-file()

Accepted values: string (filename)
Default: none

Description: Specifies a file containing Diffie-Hellman parameters, generated using the openssl dhparam utility. Note that &abbrev; supports only DH parameter files in the PEM format. If you do not set this parameter, &abbrev; uses the 2048-bit MODP Group, as described in RFC 3526.
