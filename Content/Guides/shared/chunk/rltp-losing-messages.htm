In the following cases, it is possible to lose log messages even if you use :

  • If you use together with the non-reliable disk-buffer option, it is possible to lose logs.

  • When sending logs through a relay that is using the non-reliable disk-buffer option, it is possible to lose logs if the relay crashes.

  • When sending logs through a relay that is using the non-reliable disk-buffer option, it is possible that logs are duplicated if the relay crashes, or it is stopped.

  • If the underlying disk system of fails to write the log messages to the disk, but it does not return a write error, or some other hardware or operating-system error happens.
