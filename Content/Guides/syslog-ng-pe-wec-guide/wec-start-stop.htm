Start/stop &wec;

To start and stop the &wec; tool manually, use the following commands:

systemd service for systemd-based systems:

  • Start &wecabbrev;: systemctl start syslog-ng-wec

  • Stop &wecabbrev;: systemctl stop syslog-ng-wec

SysV init for SysV-based systems:

  • Start &wecabbrev;: /etc/init.d/syslog-ng-wec start

  • Stop &wecabbrev;: /etc/init.d/syslog-ng-wec stop

To start &wecabbrev; in the foreground, execute:

wec -c /path/to/wec.yaml
