Start/stop

To start and stop the tool manually, use the following commands:

systemd service for systemd-based systems:

  • Start : systemctl start syslog-ng-wec

  • Stop : systemctl stop syslog-ng-wec

SysV init for SysV-based systems:

  • Start : /etc/init.d/syslog-ng-wec start

  • Stop : /etc/init.d/syslog-ng-wec stop

To start in the foreground, execute:

wec -c /path/to/wec.yaml
