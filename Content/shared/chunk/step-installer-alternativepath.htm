• Installation path: Enter the path to install to. This is useful if you intend to install without registering it as a service, or if it cannot be installed to the default location because of policy compliance reasons. If no path is given, is installed to the default folder.

    Installation path

    		NOTE:

    When installing to an alternative path on AIX, HP-UX, or Solaris platforms, set the CHARSETALIASDIR environmental variable to the lib subdirectory of the installation path. That way can find the charset.alias file.
